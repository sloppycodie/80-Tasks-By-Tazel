# 80 Tasks By Tazel

A collection of 80 C programming tasks assigned by **Tazel sir** as part of coursework at **Hajee Muhammad Danesh Science and Technology University (HSTU)**.

**Jibon Chandra Barman**

## 📖 About

This repository contains solutions to 80 individual programming tasks (`T1.c` through `T80.c`), covering a broad range of foundational C programming concepts as part of a university course.

> ⚠️ **Note:** These solutions are written for learning purposes. If you're a fellow student, please don't just copy-paste the code — analyze it, understand the logic, and write your own version. Copying without understanding defeats the purpose of the exercise.

## 📁 Structure

Each task is a standalone `.c` file, numbered sequentially:

```
T1.c
T2.c
T3.c
...
T80.c
```

## 🛠️ Requirements

- A C compiler (GCC recommended)

## ▶️ How to Run

Compile and run any individual task using GCC:

```bash
gcc T1.c -o T1
./T1
```

Replace `T1` with the number of the task you want to run.

## 📚 Topics Covered

The tasks span core C programming fundamentals typically covered in an intro-to-programming course, including (but not limited to):

- Variables, data types, and operators
- Conditional statements and loops
- Functions and recursion
- Arrays and strings
- Pointers
- Basic file handling
